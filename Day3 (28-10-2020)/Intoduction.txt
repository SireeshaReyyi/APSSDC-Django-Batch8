Architecture of Django:
---------------------------

MVC - Model View Controller

Model - Responsible for Database actions

View - Responsible for User Interface

Controller -  Responsible for Request-Response cycle

client - server

-> Django follows MVT architecture:

MVT - Model View Template

	                MVC               MVT

Database                Model            Model
User Interface          View             Template
Controller              Controller       View
